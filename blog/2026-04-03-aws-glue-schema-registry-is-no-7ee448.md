# AWS Glue Schema Registry is now available in three more AWS regions

<p>You can now use the&nbsp;<a href="https://docs.aws.amazon.com/glue/latest/dg/schema-registry.html">AWS Glue Schema Registry</a>, a serverless and free feature of&nbsp;<a href="https://aws.amazon.com/glue/">AWS Glue</a>, in the Asia Pacific (Jakarta), Europe (Spain), and Europe (Zurich) regions to validate and control the evolution of streaming data using registered Apache Avro, JSON, and Protobuf schema formats.</p> 
<p>The Schema Registry acts as a centralized repository for managing data format and structure between decoupled applications in data streaming systems. By using it, you can eliminate data validation logic and cross-team coordination, improve streaming data quality, and reduce downstream application failures. Through Apache-licensed serializers and deserializers, the Schema Registry integrates with C# and Java applications developed for Apache Kafka/<a href="https://aws.amazon.com/msk/">Amazon Managed Streaming for Apache Kafka</a>,&nbsp;<a href="https://aws.amazon.com/kinesis/data-streams/">Amazon Kinesis Data Streams</a>, Apache Flink/<a href="https://aws.amazon.com/kinesis/data-analytics/">Amazon Kinesis Data Analytics for Apache Flink</a>, and&nbsp;<a href="https://aws.amazon.com/lambda/">AWS Lambda</a>.&nbsp;</p> 
<p>To get started, visit the <a href="https://docs.aws.amazon.com/glue/latest/dg/schema-registry-gs.html">AWS Glue Schema Registry</a> documentation. For a full list of AWS Regions where AWS Glue Schema Registry is available, see the <a href="https://builder.aws.com/build/capabilities/explore?f=eJxVjjEPgjAQhf9K82YYXEkcWDSuMjhwDBVObKyUtAdJQ_jv1g4mbu9977vkNgS23AsPVx6NmwIqtIQllKyDlAdCh-Ln1Ks2Vt-NNRKTWE9R_aECs3cze4knY4V9cjbCTKgIzhMKgqTSdimM35DWNa_1rVFnu7Bq-ie_tfLpmyA-5ptXVh6sZfF8GTJzmR0Je7dj_wBrpUH1&amp;tab=service-feature">AWS Regional Services List</a>.</p>
