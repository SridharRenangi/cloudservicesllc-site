# Accelerate serverless application development with new SAM Kiro power

<p>AWS announces the AWS Serverless Application Model (SAM)&nbsp;Kiro power, bringing serverless application development&nbsp;expertise&nbsp;to agentic AI development in Kiro. With this power, you can build, deploy, and manage serverless applications with AI agent-assisted development directly in your local environment.</p> 
<p>SAM is an open-source framework that simplifies building serverless applications on AWS.&nbsp;SAM&nbsp;Kiro power dynamically loads relevant guidance and development&nbsp;expertise&nbsp;the AI agent needs to build serverless applications.&nbsp;This includes initializing SAM projects, building and deploying applications to AWS, and locally testing Lambda functions.&nbsp;The power&nbsp;supports event-driven patterns with Amazon&nbsp;EventBridge, Amazon Managed Streaming for Apache Kafka (MSK),&nbsp;Amazon Kinesis, Amazon DynamoDB&nbsp;Streams,&nbsp;and Amazon Simple Queue Service (SQS), while covering security best practices for IAM policies. Built-in guidance enforces&nbsp;use of&nbsp;SAM resources and&nbsp;Powertools&nbsp;for AWS Lambda&nbsp;for observability&nbsp;and&nbsp;structured logging by default, ensuring&nbsp;best practices from the start.&nbsp;This guidance accelerates your journey from concept to production, whether&nbsp;building static websites with API backends, event-driven microservices, or full-stack applications.</p> 
<p>The SAM Kiro Power is available today with&nbsp;one-click installation from the <a href="https://kiro.dev/launch/powers/aws-sam/" target="_blank">Kiro IDE</a> and the <a href="https://kiro.dev/powers/" target="_blank">Kiro Powers</a> page. Explore the power on <a href="https://github.com/kirodotdev/powers" target="_blank">Github</a>&nbsp;or visit the&nbsp;<a href="https://docs.aws.amazon.com/serverless-application-model/latest/developerguide/what-is-sam.html" target="_blank">developer guide</a>&nbsp;to learn more about SAM.</p>
