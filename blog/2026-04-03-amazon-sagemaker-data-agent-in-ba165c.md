# Amazon SageMaker Data Agent introduces charting capabilities and support for materialized views

<p>Amazon SageMaker Data Agent now supports interactive charting, SQL analytics on Snowflake data sources, and materialized view management in Amazon SageMaker Unified Studio notebooks. Data Agent now provides a complete analytics workflow that goes beyond code generation, enabling you to explore AWS and external data sources, visualize results, and optimize query performance, all with natural language prompts.</p> 
<p>You can ask "plot monthly revenue trends by region for 2025" and Data Agent generates an interactive chart directly in your notebook, where you can hover over data points, and modify without writing code. When your analysis spans AWS and Snowflake, you can query Snowflake tables through external connections and join them with your AWS Glue Data Catalog data in a single prompt. Additionally, you can ask "analyze my notebook and suggest which queries would benefit from materialized views" and the agent recommends optimizations based on your query patterns, creates the views, and sets refresh schedules.</p> 
<p>To get started, open a notebook in your <a href="https://aws.amazon.com/sagemaker/unified-studio/">SageMaker Unified Studio</a> project and use the Data Agent chat panel. These features are available in all AWS Regions where Amazon SageMaker Unified Studio is supported. To learn more, see <a href="https://docs.aws.amazon.com/sagemaker-unified-studio/latest/userguide/sagemaker-data-agent.html">SageMaker Data Agent </a>in the <a href="https://alpha.www.docs.aws.a2z.com/sagemaker-unified-studio/latest/userguide/sagemaker-data-agent.html">SageMaker Unified Studio User Guide</a>.</p>
