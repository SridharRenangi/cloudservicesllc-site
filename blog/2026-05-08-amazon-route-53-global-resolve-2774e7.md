# Amazon Route 53 Global Resolver now lets you add and remove AWS Regions for anycast DNS resolution

<p>Amazon Route 53 Global Resolver now lets you add and remove AWS Regions for anycast DNS resolution, giving you flexible control over where your DNS queries are resolved. This allows you to easily expand Global Resolver coverage as your organization grows or adjust regional deployment to meet compliance requirements.<br /> <br /> Global Resolver provides anycast DNS resolution for public internet domains and private Route 53 hosted zones from any location, along with DNS query filtering and centralized logging. With this update, you can dynamically adjust which AWS Regions participate in anycast resolution without recreating your Global Resolver configuration.<br /> <br /> This capability is available at no additional cost in all AWS Regions where Route 53 Global Resolver is supported. To get started, see the Route 53 Global Resolver <a href="https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/gr-what-is-global-resolver.html" target="_blank">documentation</a>. For regional availability, see the Route 53 Global Resolver Region <a href="https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/gr-what-is-global-resolver.html#regions" target="_blank">list</a>. For pricing, see Amazon Route 53 <a href="https://aws.amazon.com/route53/pricing/" target="_blank">pricing</a>.</p>
