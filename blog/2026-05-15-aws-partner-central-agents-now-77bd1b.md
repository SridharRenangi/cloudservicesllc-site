# AWS Partner Central agents now accelerates opportunity creation

<p>Today, AWS announces that the AWS Partner Central agents now accelerate opportunity creation through natural language conversation. <a href="https://aws.amazon.com/about-aws/whats-new/2026/03/aws-partner-central-agents-accelerate-co-sell/">AWS Partner Central agents</a>, released on March 16, 2026, are AI-powered capabilities built on Amazon Bedrock AgentCore that help partners surface pipeline insights, advance deals with next-step recommendations, and identify funding opportunities. With this update, partners create opportunities through a short conversation instead of completing a multi-step form, so partner sales teams spend less time on data entry and more time selling.<br /> <br /> Partners describe a deal in natural language, upload meeting notes, proposals, or call transcripts (PDF, DOCX, Excel, TXT), or clone an existing opportunity. The agent extracts the information, enriches customer details, and recommends improvements — such as adding missing context, correcting field values, or strengthening the business problem statement — so partners submit higher-quality opportunities, improve pipeline hygiene, and shorten sales cycles. Partners use the feature in the AWS Console through Amazon Q chat, and programmatically through Model Context Protocol (MCP), so sales teams create opportunities from their existing tools.<br /> <br /> AWS Partner Central agents are available in all commercial AWS Regions. To learn more about agentic capabilities in <a href="https://aws.amazon.com/partners/partner-central/">AWS Partner Central</a>, review this <a href="https://aws.amazon.com/blogs/apn/introducing-aws-partner-central-agents/">blog</a>. Partners can start using agents by visiting AWS Partner Central in the AWS console and accessing opportunities, after reviewing the <a href="https://docs.aws.amazon.com/partner-central/latest/sales-guide/partner-cosell-agent.html">agents guide</a>, and to integrate agents into your existing tools, visit the <a href="https://docs.aws.amazon.com/partner-central/latest/APIReference/partner-central-mcp-server.html">Partner Central agents MCP server guide</a>.</p>
